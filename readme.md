## Install

### Use `npm`

```shell
npm install -g skylor-cli
```

### Use `yarn`

```shell
yarn global add skylor-cli
```

## Usage

```shell
// 初始化 skylor
skylor init

// 查看版本号
skylor --version // or skylor -V

// 查看帮助
skylor --help

// 查看某个命名的帮助
skylor init --help
```
