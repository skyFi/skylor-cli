## Install

### Use `npm`

```shell
npm install -g skylor-cli
```

### Use `yarn`

```shell
yarn global add skylor-cli
```

## Usage

```shell
skylor init <type> ./path/to/my-project
```
### type

生成的项目类型，有以下选项：

#### NodeApi
#### Socket
#### Micro
#### ReactAdmin
#### React
#### Vue
#### ReactNative
